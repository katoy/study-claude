# ToDo アプリ テストケース

## テスト実施方法

- ブラウザで `index.html` を直接開いて手動テストを行う
- 各テストケースの実施前に、特に記載がない限り localStorage をクリアして初期状態にする
- 「現在日時」に依存するテストは、テスト実施日に合わせて期待値を読み替える

---

## 1. 初期表示

### TC-1.1: 初期状態の画面表示

- **前提**: localStorage にデータなし
- **手順**: index.html を開く
- **期待結果**:
  - タイトル「TODOリスト」が表示される
  - [新規]ボタンが表示される
  - タブ「すべて」「未完了」「完了済」が表示される
  - 「すべて」タブがアクティブ状態（視覚的に区別される）
  - [完済を削除]ボタンが表示される
  - セクション（本日中・明日まで・それ以外）は表示されない（アイテムなしのため）

### TC-1.2: 既存データの読み込み

- **前提**: localStorage に以下のデータを手動設定
  ```json
  [{"id":"test-1","title":"テストタスク","detail":"","dateType":"none","deadline":null,"completed":false,"createdAt":"2026-03-14T00:00:00"}]
  ```
- **手順**: index.html を開く
- **期待結果**:
  - 「テストタスク」が「それ以外」セクションに表示される
